# Github_fetch_issues

## Prerequisites:
* python 3.6+
* Virtual Environment

## virtual environment:

* python -m venv venv
## activate:
* source venv/bin/activate(linux)
* venv\Scripts\activate(windows)

## Installation and running:
* cd Github_fetch_issues
* python -m pip install --upgrade pip
* pip install -r requirements.txt
* python manage.py runserver
